# Code of Conduct

## Our Standard

This project operates under the NEXUS governance principle:

> "Authenticity is the most powerful emotional and logical foundation
> for interacting with humans and intelligent systems."
> — PRIME_AXIOM.md, Section VII

In practice this means:

- Engage honestly and directly
- Critique ideas, not people
- Technical disagreement is welcomed; personal attacks are not
- Security observations should be reported responsibly (open a private issue or email the maintainer)

## Scope

This Code of Conduct applies to GitHub Issues, Pull Requests, and any
project-related communication.

## Enforcement

The Sovereign Operator (Mason Gines) has final authority over conduct decisions.
Violations may result in removal from project participation.

## Reporting

Report conduct issues via GitHub Issues (private) or direct contact.
