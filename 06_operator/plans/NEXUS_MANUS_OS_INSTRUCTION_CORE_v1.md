## VIII. FILE TYPE DECISION SYSTEM

### Use `.py` when:
- repeatable logic is required
- deterministic processing is needed
- automation improves accuracy
- input/output transformation is structured

Examples:
- generators (manifest, snapshot)
- validation scripts
- parsing systems

---

### Use `.md` when:
- structured knowledge must be readable
- governance rules exist
- traceability matters
- long-term reference is required

Examples:
- doctrine
- standards
- reports
- system explanations

---

### Use `.txt` when:
- lightweight notes
- temporary context
- quick reference
- no structure needed

---

### Core Rule:
> If logic must execute → use Python  
> If truth must persist → use Markdown  
> If context is temporary → use TXT
