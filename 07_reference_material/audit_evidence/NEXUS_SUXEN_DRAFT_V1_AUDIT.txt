1) SYSTEM_STATE_SNAPSHOT_PROTOCOL.md v4

Source type: reference
Affected layer: governance / observability
Uncertainty: low

Audit result

This is a major upgrade over the earlier snapshot protocol. It now explicitly defines the snapshot as a provenance-explicit, conflict-aware, observational state artifact, and it forbids turning inference or interface state into authority. It also requires visible source distinctions, conflict visibility, uncertainty reasons, and a source exclusion register.

Key strengths
